# angular-sockjs-clojure

A chat application written in Clojure and AngularJS

## Usage

Compile client side code to javascript once:
```
lein cl2c once
```
or automatically recompile the file on changes:
```
lein cl2c auto
```

Start the server:
```
lein run
```

Enjoy!
